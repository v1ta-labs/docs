v1ta is built as a Solana program using the Anchor framework. This document provides a comprehensive overview of the technical architecture and design decisions.

## System Overview

v1ta Protocol consists of a Global State account, VUSD Mint, Stability Pool, and multiple user Position accounts (PDAs) and Stability Deposit accounts (PDAs). The protocol connects to a Pyth Oracle for SOL/USD price feeds.
